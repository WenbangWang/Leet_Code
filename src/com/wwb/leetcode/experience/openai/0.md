今年Q1面的，面的是 Algorithm & Data Structures Interview，基本就是利口2502，然后变了一下

搞一个内存分配器，要能分配和释放内存，做过类似的题或者上学的时候写过的话应该知道接口其实特别简单。


跟2502有点不一样，内存并没有ID，但是有地址的概念，基本上就是内存块的base+offset就是内存的起始和结束。面试官会给一堆测试，主要就测一下malloc和free是不是符合预期

因为有地址的存在，一通malloc和free过后难免有很多内存碎片，所以之后需要进行内存整理，基本上就是想办法把内存挪到一起，空出大块的内存。这一块没写好加上时间来不及了于是就挂了。
